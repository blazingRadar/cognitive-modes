# Publication Manifest

This repository is a public research-notes archive for preliminary prompt-framing experiments in LLM code review.

## Included

- Bounded README and review path.
- A concise synthesis note.
- Citation metadata and archive status.

## Not Included

- Raw transcripts.
- Tested source files.
- Scored result tables.
- Prompt-run metadata.
- Statistical replication across codebases, prompts, or model versions.

## Claim Boundary

The repository supports a preliminary case-study claim: in the observed runs, changing the framing of a code-review prompt changed the kind of output the model produced.

The repository does not support broad claims about all models, a validated mode taxonomy, production calibration, or stable provider-specific behavior.

## Hygiene Notes

- No API keys or environment files are included.
- No private raw-data links are used as evidence.
- The public artifact does not claim independent reproducibility because raw transcripts are not included.
- External links are limited to public author and related-repository links.
